## 左值引用右值引用

- 左值：可以取地址的，有名字的，非临时的

- - 变量
  - 函数返回的引用

- 右值：不能取地址的，没有名字的，临时的

- - 立即数
  - 函数返回值

之前的常规引用都是左值引用

右值引用绑定到右值后本来会被销毁的右值的生存期会延长至与绑定到它的右值引用的生存期。

```C++
// 带右值引用参数的拷贝构造函数

Stack(Stack &&src)
  :msize(src.msize), mtop(src.mtop)
{
  cout << "Stack(Stack&&)" << endl;

  /*此处没有重新开辟内存拷贝数据，把src的资源直接给当前对象，再把src置空*/
  mpstack = src.mpstack;  
  src.mpstack = nullptr;
}
```
